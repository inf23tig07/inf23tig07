Apresentação do Projeto
Este projeto tem como objetivo criar um site informativo sobre Basquetebol, cobrindo vários aspectos do desporto, desde as regras básicas até aos principais torneios e estrelas do jogo. O site terá quatro páginas estáticas, utilizando HTML5, CSS3, JavaScript e XML para garantir uma estrutura bem organizada, visualmente atraente e funcional.

Estrutura do Site
Página Inicial (index.html)

Apresentação geral do Basquetebol.
Breve histórico do desporto.
Ligações para as outras páginas do site.
Regras do Jogo (regras.html)

Explicação das principais regras do Basquetebol.
Tabela comparativa entre regras de diferentes ligas (NBA, FIBA).
Torneios (torneios.html)

Informações sobre os principais torneios de Basquetebol (NBA, EuroLeague, etc.).
Tabela de vencedores dos últimos anos.
Jogadores Famosos (jogadores.html)

Perfis de jogadores icônicos, com biografias e estatísticas.
Galeria de imagens dos jogadores.
XML Document (jogadores.xml)

Contém dados sobre jogadores famosos, incluindo nome, equipa, posição, e estatísticas principais.
Detalhes Técnicos
HTML
Utilização correta de marcação semântica.
Inclusão de tabelas, listas, imagens, e formulários.
Exemplos:
Tabela com utilização de thead, tbody, tfoot, rowspan, colspan.
Listas ordenadas, não ordenadas e de definições, com aninhamento.
Elementos de destaque (em, strong, mark).
CSS
Estilos externos, sem utilização de inline.
Exemplos de seletores simples, pseudo-elementos e combinadores.
Propriedades de texto, fonte, fundo, listas, formatação de tabelas.
Manipulação de caixas de elementos HTML.
Utilização de flexbox, transições, transformações e animações.
Responsividade para diferentes dimensões de ecrã (media queries).
JavaScript
Manipulação de elementos HTML em resposta a eventos.
Integração dinâmica de conteúdo XML.
Exemplo: Conversão de dados XML em tabela HTML.
XML
Documento XML com dados sobre jogadores.
Schema XML para validação.
